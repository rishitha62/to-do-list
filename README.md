# To-Do List Application

A simple, intuitive, and fully client-side to-do list application built with vanilla HTML, CSS, and JavaScript.

## Features

### Core Functionality
- Add Tasks: Quickly add new tasks to your list
- View Tasks: See all your tasks in an organized list
- Mark Complete: Check off tasks as you complete them
- Delete Tasks: Remove tasks you no longer need
- Clear Completed: Batch delete all completed tasks
- Task Statistics: View counts of total, active, and completed tasks

### Technical Features
- Instant Updates: All changes happen immediately without page reloads
- Local Storage: Tasks persist across browser sessions
- Responsive Design: Works seamlessly on desktop and mobile devices
- Accessible: Keyboard navigation and ARIA labels supported
- Modern UI: Clean, gradient-based design with smooth animations
- No Backend: Fully client-side, no server or database required
- No Authentication: Start using immediately, no login needed

## User Stories Implemented

1. Add Tasks: As a user, I want to add new tasks, so that I can keep track of things I need to do.
2. View Added Tasks: As a user, I want to view all my added tasks, so that I can see what I need to accomplish.
3. Mark Tasks as Completed: As a user, I want to mark tasks as completed, so that I can track my progress.
4. Delete Tasks: As a user, I want to delete tasks, so that I can remove items I no longer need to track.
5. User-Friendly Interface: As a user, I want a simple and intuitive interface, so that I can manage my tasks easily.
6. No Login or Registration: As a user, I want to use the task list without having to log in or register.
7. No Backend Requirement: As a user, I want my interactions to be managed instantly on the interface.
8. Immediate Effect: As a user, I want changes to appear instantly without needing to refresh the page.
